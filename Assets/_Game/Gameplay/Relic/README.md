# FeatureName

Esta pasta serve como template para novas features.

**Como usar:**
- Renomeie esta pasta para o nome da sua feature (ex: Player, Menu, Enemy).
- Use as subpastas para organizar scripts, prefabs, UI, materiais e áudio relacionados à feature.

**Subpastas:**
- Animations: Animações da feature
- Audio: Sons e músicas da feature
- Materials: Materiais específicos
- Prefabs: Prefabs usados pela feature
- Scripts: Scripts C# da feature
- UI: Elementos de interface da feature
